# AI Chat Exporter - Installation Details

## Production Installation (User)
1. Navigate to the Chrome Web Store (Link pending).
2. Click **"Add to Chrome"**.
3. Pin the extension to your browser toolbar for quick access.

## Developer Installation (Manual Load)
If you are developing the extension or want to test the latest unreleased version:

1. **Clone or Download the Repository:**
   Download the latest source code to a directory on your machine.

2. **Open Chrome Extensions Page:**
   Open Chrome and navigate to `chrome://extensions/`.

3. **Enable Developer Mode:**
   In the top right corner of the Extensions page, toggle the **"Developer mode"** switch to ON.

4. **Load Unpacked Extension:**
   - Click the **"Load unpacked"** button that appears in the top left.
   - Navigate to the directory where you cloned/downloaded this repository (the folder containing `manifest.json`).
   - Select the folder.

5. **Verify Installation:**
   The "AI Chat Exporter" icon should now appear in your browser toolbar. Clicking it on a supported AI chat page (ChatGPT, Claude, Gemini, AI Studio) will open the popup.

## Supported Platforms
- ChatGPT (`chatgpt.com`)
- Claude (`claude.ai`)
- Google Gemini (`gemini.google.com`)
- Google AI Studio (`aistudio.google.com`)
